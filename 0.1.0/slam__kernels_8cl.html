kernel void splitPC8D_octomap ( global float8 *  pc8d,
global float *  pc3d,
global uchar *  rgb 
)

Splits an 8-D point cloud into 3-D coordinates (in meters) and 8-bit RGB values.

Note
The global workspace should be one-dimensional. The x dimension of the global workspace, \( gXdim \), should be equal to the number of points in the point cloud. The local workspace is irrelevant.
Parameters
[in]pc8darray with 8-D points (homogeneous coordinates + RGBA values).
[out]pc3darray with 3-D coordinates (in meters).
[out]rgbarray with 8-bit RGB values.
